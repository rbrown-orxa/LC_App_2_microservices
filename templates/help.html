Instruction Manual



Description: OrxaGrid SolarPV tool is used to size your proposed SolarPV Installation. This is for use by building owners who would like to save money on their energy bills by installing Solar Panels on their roof. This tool will calculate the optimal size of the proposed SolarPV installation to maximize the financial return.

Site Url: https://derapp.orxa.io/

Application host: Azure Web App service

Instructions:

  1. Login: The app is currently available free of use. There is only single user authentication.
    Click the site URL or paste the site URL on browser and press ENTER key. Key in username =     admin and password = password and click Log In button. Then page is directed to login page.



  1. Location Selection: Move the scroll button to the bottom of the screen so as place google maps covering the entire page. Select the Enter the location Text box and enter the name of city your address just like you use to search location using google maps and select the one which is closest address of your location. Then location latitude and longitude are auto populated in the form input Latitude & longitude. Then scroll back the button to the top so that the form is fully visible.



  1. Form inputs: The form inputs currently are populated with default values in currency GBP. The user can enter the values in the same currency for Cost per KWp which is the installation and commissioning of 1KW of Solar Panel. Import Cost is the price customer pays to the utility per KWh for energy consumption. Export Price is the price the customer gets by selling per KWh of energy. The other inputs are expected life of the solar panel which be around 20-25 years. The roof size for installing the panel in m^2


    Finally the user can upload load file only csv format by clicking choose file or just drag and drop the file. The load file can be accepted in variety of formats as a) Single Column Time b) Single Column Period c) Row per daytime d) Row per Day period. Snapshot of the various inputs file formats are pasted. After uploading the file the click the submit button to run the solar optimizer tool until the Loading button stops and the result page appears. Download Sample file
